  
  I- prerequis de l'installation: 


- Disposé d'un ordinateur et d'une connexion à internet;

- installer un IDE par exemple VScode;

- installer wampserver ou xammp

- installer laravel dans votre machine 

- cloner le projet sur le depot à distant de Github 

- se deplacer sur la branche develop et faire un git pull -u origin master;


II- Demmarage du serveur 

dans vscode vous devez demmarer le serveur a l'aide de la commande php artisan serve. 


III-lancement des test

Vous lancer les testes à l'aide de la commande php artisan test 

Url de la documentation du projet accesible en http://localhost8000/api/documentation
